# Student Social Media Addiction – BI Analysis

## Project Overview
This project analyzes social media usage patterns and addiction risk among students.
The goal was to build an end-to-end BI project using SQL for data preparation
and Power BI for data visualization.

## Tools Used
- PostgreSQL
- SQL (data cleaning & transformations)
- Power BI
- Docker
- GitHub

## Data Pipeline
1. Raw data loaded into PostgreSQL
2. Data cleaned and transformed using SQL
3. Final dataset connected to Power BI
4. Interactive dashboard created

## Dashboard Insights
- Total number of students: 705
- Average daily social media usage: ~4.9 hours
- Average sleep duration: ~6.9 hours
- Majority of students fall into the *Medium addiction risk* category
- Younger age groups tend to spend more time on social media
